#### 每种字体因为设计的原因,同为 20px,可能看起来大小不一样

#### 通常情况下,文字相对字体设计框.并不是垂直居中的,通常都靠下一些.

#### case:

- 父元素内,不同 font-size 的元素居中对齐

```js
// html
<div>
    <span></span>
    <span></span>
</div>
// css
div {
    font-size: 0
}
div:nth-child(1) {
    font-size: 40px;
    vertical-align:middle;
}
div:nth-child(2) {
    font-size: 20px;
    vertical-align:middle;
}
```
