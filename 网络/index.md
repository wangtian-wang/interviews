## 对称加密和非对称加密的区别？？

|      | 对称加密                                       | 非对称加密                                                                                                   |
| :--- | :--------------------------------------------- | :----------------------------------------------------------------------------------------------------------- |
| 定义 | 加密和解密使用的是同一个秘钥                   | 公钥加密，私钥解密                                                                                           |
| 优点 | 算法公开、计算量小、（加密）速度快、效率高     | 时间长、速度慢、适合少量数据                                                                                 |
| 缺点 | 秘钥安全传递有风险，密文容易被截获，安全性较低 | 私钥是基于算法生成的随机数，私钥通过算法生成公钥，私钥到公钥的过程是单向的，只有私钥能推导出公钥。安全性高。 |
| 举例 | Blowfish、IDEA、RC4、RC5、RC6 和 AES           | RSA、DSA（数字签名用）                                                                                       |
