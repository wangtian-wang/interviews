#### 第三方库的组件按需加载

- `element-ui`

  - `babel-plugin-component`

    > 1: 分析我们从`element`中导出的组件,判断我们是否使用了该组件

    > 2: 帮我们去 node_modules 里面加载对应的组件的 js 和 css

    > 整个实现的思路是基于 ast 的词法分析
    > 因为我们使用的第三库都是打包好的文件 js css 文件是分开的.引入组件后,需要去找到组件对应的文件
