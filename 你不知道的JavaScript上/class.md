## 继承的方式

#### 显示混入

将一个类中的属性或者方法 手动的复制到另外一个类中 （将类看做对象 ）**_复制的是引用 而不是对象本身_**

#### 混合复制

将类 A 中的属性， 在类 B 中如果不存在 就混合（复制)到类 B 中
